== README

## Description of the appliacation

This is demo application to demostrate usage of the datetimepicer-rails gem. It is for the versions 4.0.0+ of the gem.

## Ruby on Rails

 - Ruby version: ruby-2.2.0,
 - Rails version: rails4.2

## System dependencies

 - datetimepicer-rails (versions 4.0.0+)
 - bootstrap-datetimepicker (version 4.0.0)
 - Simpleform 3.1+
 - Momentjs-rails 2.9+
